# web-intro-edit5. Báo Cáo Kết Quả Thực Hành - Tab Network

Báo Cáo Thực Hành: Giám Sát Thẻ Network

Môn học: Phát triển Ứng dụng Web Bài thực hành: Sử dụng Trình duyệt DevTools Năm thực hiện: 2026
📝 Ghi chú mục tiêu: Bảng nhật ký dưới đây ghi lại danh sách các tài nguyên được hệ thống tải về khi truy cập trang mẫu. Việc giám sát giúp phân tích rõ thực thể nào đưa ra yêu cầu (Client Request) và thực thể nào chịu trách nhiệm phản hồi thông tin dữ liệu (Server Response).
Bộ lọc nhanh:
Tên Tài Nguyên (Name) Loại (Type) Trạng Thái Bên Gửi Yêu Cầu (Client Request) Bên Trả Phản Hồi (Server Response)
index.html document 200 OK Trình duyệt nhập URL (Người dùng kích hoạt) Hệ thống Web Server (Mã nguồn Host)
style.css stylesheet 200 OK Trình duyệt phân tích HTML bắt gặp thẻ <link> Máy chủ CDN lưu trữ file CSS tĩnh
app.js script 200 OK Trình duyệt đọc HTML bắt gặp thẻ <script> Hệ thống Web Server (Mã nguồn Host)
logo.png image 200 OK Trình duyệt đọc DOM bắt gặp thẻ <img src="..."> Kho lưu trữ dữ liệu hình ảnh (Cloud Storage)
banner.jpg image 200 OK Trình duyệt đọc thuộc tính background-image của CSS Kho lưu trữ dữ liệu hình ảnh (Cloud Storage)

Sinh viên thực hiện hoàn thành biểu mẫu báo cáo kỹ thuật Network Tab
